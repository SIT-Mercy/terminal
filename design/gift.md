# 礼品管理 Gift

用户只能线下使用积分(point)兑换礼品(redeem gift)。
操作员可以在同一局域网下，使用node app连接启动了terminal的主机，执行所有功能。

## 添加礼品

- [必选] 操作员 operator
- [必选] 礼品名称 name
- [必选] 所需积分 pointRequirement
- [可选] 礼品介绍 descriotion
- [可选] 礼品图片 image

## 删除礼品

- [必选] 操作员 operator

## 修改礼品信息

- 礼品名称 name
- 礼品简介 descriotion
- 礼品图片 image
- 所需积分 pointRequirement
